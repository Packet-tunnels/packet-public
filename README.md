# Packet - Secure Network Tunnel

Packet is a modern, high-performance network tunnel designed for privacy and security.

## Support
If you encounter any issues or have questions regarding Packet, please contact our support team:
- **Email:** support@packetapp.net
- **GitHub Issues:** Use the issues tab in this repository.

## Frequently Asked Questions
### How do I connect?
Add your server URL and shared secret in the app settings, then tap the "Connect" button on the dashboard.

### Is my data secure?
Packet uses advanced encryption protocols to secure your traffic. We do not bundle analytics or tracking SDKs.
